## Contributing
みなさんの貢献は大いに歓迎します。  
Goalousを共に改善していきましょう。  
Goalousに貢献したいという素晴らしいマインドを持っている方は以下の手順で開発したものを送って下さい。

1. Forkする  
Githubページ上右上の「Fork」をクリックしてリポジトリをコピーします。
1. ForkしたリポジトリをCloneする  
1. 作業ブランチを切る  
1. コミットする
1. pushする
1. Pull Requestする(マージ対象はdevelop)
